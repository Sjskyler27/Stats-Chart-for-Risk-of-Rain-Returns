This program is meant to display some of the stats from risk of rain returns in a digestible format.
Use the arrow keys to cycle through different stats.
When you first run the program it will create a config file. The config file will save the names of the profile in steam user data.
If you run into issues with loading the save data you can edit the path and the users in the config file. 
There are also settings to adjust how the charts are laid out, the program will atempt to get it right but it may need some tweaking to make it look good.
you can use the settings button in the program to find out what values work for your screen.
